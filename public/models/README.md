将你选中的海上风机资产导出为 GLB 文件，并放到下面这个路径：

`public/models/offshore-wind-turbine.glb`

当前代码会优先加载这个文件：

- 存在该文件：使用真实 GLB 风机资产
- 文件不存在或加载失败：自动回退到程序化风机占位模型

建议优先选择包含完整塔架、机舱、叶片的海上风机模型。

当前选定来源：

- Sketchfab: Generic Wind Turbine (V136 125.5h 145d)
